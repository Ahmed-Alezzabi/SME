# منظومة إدارة التدريب والمشاركين بمركز أعمال طرابلس
## Tripoli Business Center Training & Certificates Management System

منظومة ويب متكاملة ومخصصة لـ **مركز أعمال طرابلس** التابع لـ **البرنامج الوطني لدعم المشروعات الصغرى والمتوسطة (وزارة الاقتصاد والتجارة - ليبيا)**.
الموقع الرسمي للمؤسسة: [https://sme.gov.ly](https://sme.gov.ly)

تم تصميم وتطوير المنظومة بالكامل باللغة العربية مع دعم فني متكامل للاتجاه من اليمين إلى اليسار (RTL) بهوية بصرية مستوحاة من الموقع الرسمي للمؤسسة ووزارة الاقتصاد.

---

## 🚀 المميزات التقنية للمنظومة

### 1. إدارة البرامج التدريبية والتنموية
- إضافة وتعديل وحذف وأرشفة البرامج التدريبية.
- نسخ ومضاعفة البرامج السابقة بنقرة واحدة لتسهيل التوثيق.
- تصنيف مجدول للبرامج: السابقة، الجارية حالياً، وقيد التخطيط والمقبلة.

### 2. سجل وبيانات المشاركين ورواد الأعمال
- حصر شامل لبيانات المتدربين (الاسم الرباعي، الرقم الوطني الفريد، المهنة، جهة العمل، المدينة).
- البحث المتقدم والفرز بحسب البرنامج، نتائج التحصيل أو حالة وقرار الاجتياز والمطابقة.
- **نظام استيراد ومعالجة ذكي من Excel**: يتيح نسخ الخلايا مباشرة من الإكسل ولصقها مع المعالجة، كشف التكرار للرقم الوطني وخيارات تحديث أو دمج البيانات.
- تصدير فوري لبيانات المتدربين بصيغة كتلية ملائمة لـ Excel (CSV).

### 3. تقويم مركز الأعمال التفاعلي
- واجهة أجندة متقدمة تعرض الفترات الزمنية للبرامج.
- خيارات تصفح: شهري، أسبوعي، ويومي بالتزامن مع تواريخ التنفيذ.

### 4. كشوفات وإصدار الشهادات التدريبية الإلكترونية المعتمدة
- توليد شهادات إلكترونية عريضة (A4 Landscape) بنسق وتصميم عربي عريق.
- تعبئة آلية للبيانات: اسم المتدرب الرباعي، اسم الدورة، الساعات المعتمدة، وتواريخ الحضور.
- **رمز التحقق السريع (QR Code)**: يمكن مسحه للتحقق المباشر والآمن ومطابقة صحة الشهادة لحماية مصداقيتها أمام جهات التمويل والمصارف.

### 5. التقارير الإحصائية المتقدمة والبيانات الفنية
- كشوف حضور وغياب وتوزيع جغرافي للمنتسبين.
- رسوم بيانية توضح معدلات النجاح والنسب المئوية في مختلف البرامج.
- دعم الطباعة المباشرة لجميع التقارير والشهادات.

## 🛠️ المعايير المعمارية والبرمجية (Tech Stack)

### Frontend:
- **React 19** & **TypeScript** لضمان الأمان البرمجي وتجنب الأخطاء.
- **Tailwind CSS Utility Classes** مع خط **Cairo (عربي)** متميز ومناسب للواجهات المؤسسية.
- أيقونات تفاعلية باستخدام **Lucide React**.

### Backend API:
- خادم مخصص مبني بواسطة **Express Node.js Server**.
- دمج سلس للواجهات المترجمة والـ APIs ورسم لوحات الداتا.

### Database Persistence:
- قاعدة بيانات JSON محلية مرنة لتأمين البيانات وحفظها تلقائياً.
- **Prisma Schema** كامل متصل بـ Database **PostgreSQL** مستعد للمزامنة والتشغيل الفوري.

## 📦 تشغيل المنظومة محلياً (Run & Install Guide)

### المتطلبات المسبقة:
- مثبت حزم العمل [Node.js v18+](https://nodejs.org/)

### 1. تثبيت الاعتمادات
```bash
npm install
```

### 2. تشغيل بيئة التطوير
```bash
npm run dev
```
سيتم تشغيل واجهة المنظومة على منفذ الاتصال الافتراضي: `http://localhost:3000`

### 3. بناء نسخة الإنتاج
```bash
npm run build
```

### 4. تشغيل خادم الإنتاج الفعلي
```bash
npm run start
```

## 🐳 التشغيل السريع بواسطة دكر (Docker Configuration)

المنظومة مزودة بملف Dockerfile جاهز للتشغيل والإنتاج المباشر.

### 1. بناء صورة دكر للبرنامج:
```bash
docker build -t tripoli-sme-system .
```

### 2. تشغيل الحاوية المبرمجة:
```bash
docker run -d -p 3000:3000 --name tripoli-sme-system tripoli-sme-system
```

## 🔒 حماية الجلسة والتحكم بالصلاحيات (RBAC)

تشتمل المنظومة على محاكاة متطورة لنظام الأدوار لمركز أعمال طرابلس:
1. **مدير النظام (Admin)**: صلاحيات كاملة للتعديل والتحكم بالبيانات وإعادة الضبط.
2. **موظف التدريب (Trainer)**: إضافة وتعديل وجدولة البرامج والشهادات.
3. **موظف التسجيل (Registrar)**: تسجيل المتدربين وتقييم درجاتهم ورصد الحضور.
4. **موظف الشهادات (Certificates)**: مراجعة المستندات وإصدار وتوثيق الشهادات برمز QR.
5. **قارئ فقط (Read-Only)**: تصفح الكشوف والتقارير الإحصائية دون إمكانية التعديل.

## 📈 دليل تهيئة ونمذجة قاعدة البيانات (Prisma PostgreSQL)

تم تزويد المنظومة بملف مخطط قاعدة بيانات متكامل **Prisma Schema** يشتمل على:
- جدول البرامج التدريبية العلاقي ومخطوطة البيانات.
- جدول المشاركين المسجلين مع مفتاح خارجي (Foreign Key) متصل بالبرنامج.
- جدول سجلات الأمان والامتثال (Audit Logs) للرقابة.

الملف يقع تحت: `/prisma/schema.prisma`
